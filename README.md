## 🎯 The Goal

The main objective of this build wasn't just to make it look like the original, but to actually figure out how those high-end editorial websites handle spacing and text reveals. 

A few specific things I tackled:
* **Typography & Layout:** Getting the exact font weights, letter spacing, and grid alignment right. Minimal design is surprisingly hard because any mistake in spacing is super obvious.
* **Scroll Animations:** Used GSAP and ScrollTrigger to recreate the smooth text reveals and image scaling as you scroll down the page.
* **Desktop First:** I focused entirely on making the desktop experience and animations as perfect as possible (so definitely view this on a monitor!).

## 💻 Tech Stack

* HTML5
* CSS3 (Custom properties, Flexbox/Grid)
* Vanilla JavaScript
* GSAP & ScrollTrigger
* Locomotive Scroll
